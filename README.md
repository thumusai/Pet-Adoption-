# Pet-Adoption-
🐾 AdoptMe — Pet Adoption Webpage
A simple and heartwarming pet adoption webpage built with HTML & CSS.
Give pets like Jerry and Tom a second chance at a happy life! 🐶🐱

📸 Preview

A clean card-based layout showcasing adorable pets available for adoption.


🛠️ Built With

HTML5 — Structure of the webpage
CSS3 — Styling and layout (cards, colors, fonts)

✨ Features

🐶 Dog adoption card — Meet Jerry

🎨 Clean card layout — Easy to read and navigate
